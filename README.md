# Signiel-Hotel-Website-Project

#프로젝트 기간
22.10.17 ~ 22.11.04 

#기술 스택
1. Web Front : JSP, CSS, JavaSCript,Jquery
2. Web Server : Java(1.8.0), ApacheTomcat
3. DBMS : MySQL
4. Tool : Eclipse, HeidiSQL


#기능 구현
1. 회원가입 및 로그인, 로그아웃
2. 회원 수정 및 탈퇴
3. 아이디, 이메일 중복검사
4. 게시글 작성 및 수정,삭제
5. 게시글 검색
6. 객실 예약
7. 예약 내역 조회 및 취소
